# CharFlow Presets

A curated collection of pre-made presets designed to give your interface distinct personality and atmosphere.

## Installation

1. Navigate to the `presets` folder in this repository.
2. Select the preset you want to use.
3. Open the file, then click the **Download** button (next to "Raw" / "Copy raw file").
4. Go to [character.ai](https://character.ai).
5. Open the mod menu and go to the **Presets** category.
6. Click **Import Preset**.
7. Select the file you downloaded.
8. You're all set.
